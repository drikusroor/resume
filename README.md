# Drikus Roor Resume

This is the printable resume website of Drikus Roor of 2023/2024.

## Getting started

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run serve
```

This project was created using `bun init` in bun v1.0.0. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
